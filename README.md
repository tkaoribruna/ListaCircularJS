# Lista Circular em JavaScript
## Disciplina: Estrutura de Dados - 2026 / 1º Semestre
Este projeto é uma aplicação web simples desenvolvida com a finalidade de demonstrar o funcionamento prático de uma **Estrutura de Dados do tipo Lista Circular**.

---

## 📂 Estrutura do Projeto

* `index.html`: Interface visual com o painel de controle, campos de entrada (`inputs`) e botões de ação.
* `listaCircular.js`: Contém as classes `Nodo` e `ListaCircular`, além das funções que conectam a interface gráfica (DOM) à lógica da estrutura de dados.
* `hello.js`: Um arquivo de script secundário com testes básicos de console e validações iniciais.

---

## Linguagens Utilizadas

* **HTML5:** Estrutura e marcação da página web.
* **JavaScript (ES6):** Lógica orientada a objetos (Classes, Métodos, Arrow Functions e Manipulação de DOM).
